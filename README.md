# D0s
This script is Dos attack

## How to install it? (วิธีการติดตั้ง)
```
apt install git
```
```
apt install python2
```
```
git clone https://github.com/WeirHK/D0s/
```

## How to run it? (วิธีการรัน)
```
cd D0s
```
```
python2 Dos-WHKTH.py
```
